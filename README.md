## 📸 About This Project

## ⚙️ Installation & Setup

Follow these steps to run this project locally:

### 1. Requirements
- XAMPP (or any local server with PHP & MySQL)
- WordPress 6.x
- Web Browser

### 2. Clone the Repository

```bash
git clone https://github.com/yourusername/fashion-lover-wordpress.git
```

### 3. Setup Database

1. Open **phpMyAdmin** → `http://localhost/phpmyadmin`
2. Create a new database named `fashiondbnew`
3. Import the SQL file (if included):
   ```
   fashiondbnew.sql
   ```

### 4. Configure WordPress

1. Copy `wp-config-sample.php` → rename to `wp-config.php`
2. Update the database details:

```php
define( 'DB_NAME', 'fashiondbnew' );
define( 'DB_USER', 'root' );
define( 'DB_PASSWORD', '' );
define( 'DB_HOST', 'localhost' );
```

### 5. Move to XAMPP

Place the project folder inside:
```
C:/xampp/htdocs/fashionsite/wordpress/
```

### 6. Run the Site

Open your browser and visit:
```
http://localhost/fashionsite/wordpress/
```
